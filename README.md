# 📒 Agenda de Contatos

Aplicação Console desenvolvida em **C#** com foco no aprendizado de **Programação Orientada a Objetos (POO)**.

Este projeto permite gerenciar uma lista de contatos por meio de um menu interativo no terminal.

## Funcionalidades

* ➕ Adicionar contato
* 📋 Listar contatos
* 🔍 Buscar contato pelo nome
* ❌ Remover contato
* 🚪 Encerrar a aplicação

## Conceitos aplicados

* Classes e Objetos
* Encapsulamento
* Construtores
* Listas (`List<T>`)
* Métodos
* Estruturas de repetição (`for` e `foreach`)
* Estruturas de decisão (`if`, `switch`)
* Tratamento de exceções (`try/catch`)
* Organização do código em múltiplas classes

## Estrutura do projeto

* **Contato**: representa um contato contendo nome e telefone.
* **Agenda**: responsável por gerenciar os contatos.
* **Menu**: exibe as opções da aplicação.
* **Pausa**: centraliza a pausa entre as operações.
* **Erro**: exibe mensagens de erro.
* **Sair**: encerra a aplicação.

## Tecnologias utilizadas

* C#
* .NET
* Visual Studio
